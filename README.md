# Text Photo to PDF

This project allows users to upload an image containing text, extract the text using OCR, and export it into a PDF.

## Features
- Upload text-containing images
- Preview image
- Extract text using OCR (Tesseract / EasyOCR)
- Include or exclude original image in PDF
- Choose PDF page size (A4, Letter)
- Download generated PDF

## Deployment
Designed for deployment on Hugging Face Spaces using Streamlit.
